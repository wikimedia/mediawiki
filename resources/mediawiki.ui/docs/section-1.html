1 - Buttons

1.1 - Button styling

Basic button styling to be used on desktop skins. Can be any element. Due to a lack of a CSS reset, the exact styling of the button depends on what type of element is used.

Output

DIV.mw-ui-button

Sample HTML

<div class="mw-ui-button">DIV.mw-ui-button</div>
<button class="mw-ui-button">button.mw-ui-button</button>
<input type="submit" class="mw-ui-button" value="input[type=submit].mw-ui-button">

1.2 - Button groups

Group of buttons. Make sure you clear the floating after using a mw-ui-button-group.

Output

A
B
C
C

Sample HTML

<div class="mw-ui-button-group">
  <div class="mw-ui-button">A</div>
  <div class="mw-ui-button">B</div>
  <div class="mw-ui-button">C</div>
  <div class="mw-ui-button">C</div>
</div><div style="clear:both"></div>
