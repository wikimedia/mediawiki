2.0 - Forms

Style a compact vertical stacked form using .mw-ui-vform and the elements in divs within it.
Default styling
DIV
Example markup 
<form class="mw-ui-vform">
  <div>DIV</div>
  <label>label</label>
  <input value="input">
  <input type="checkbox" checked>
  <input type="radio" checked>
  <button class="mw-ui-button">Button in vform</button>
</form>
